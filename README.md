# ลิสต์ (List)

## ลิสต์แบบเรียงลำดับ (Ordered List)

```md
1. สวัสดี
2. ฉัน
3. ชื่อ
4. บุษบา
```

1. สวัสดี
2. ฉัน
3. ชื่อ
4. บุษบา

## ลิสต์แบบไม่เรียงลำดับ (Unordered List)

```md
* สวัสดี
* สวัสดี
* สวัสดี

ได้ทั้งสองแบบ

- สวัสดี
- สวัสดี
- สวัสดี
```

* สวัสดี
* สวัสดี
* สวัสดี

ได้ทั้งสองแบบ

- สวัสดี
- สวัสดี
- สวัสดี

## หรือจะ list ซ้อน list ก็ได้ (Nested List)

```md
1. สวัสดี
   - ฉันชื่อ
     - บุษบา
       1. ลาล่าล่าล๊า
       2. ลาล่าล่าล๊า
```

1. สวัสดี
   - ฉันชื่อ
     - บุษบา
       1. ลาล่าล่าล๊า
       2. ลาล่าล่าล๊า

# อ้างอิง (Quotes)

## อ้างอิงธรรมดา

```md
> ตัวอย่างการอ้างอิง
> นี่คือตัวอย่าง
> การอ้างอิง
```

> ตัวอย่างการอ้างอิง
> นี่คือตัวอย่าง
> การอ้างอิง

## อ้างอิงซ้อนอ้างอิง (Nested quote)

```md
> อ้างอิง
> > ซ้อนอ้างอิง
> > > ซ้อนซ้อนอ้างอิง
```

> อ้างอิง
> > ซ้อนอ้างอิง
> > > ซ้อนซ้อนอ้างอิง

## อ้างอิง Advanced


```md

> 
> 1. รายการที่ 1
> 2. รายการที่ 2
>    - รายการย่อย 1
>    - รายการย่อย 2

> console.log("advanced md");
> ```
```
